# Documentation

This directory contains detailed documentation for the Desk Rejection Detection system.

## Available Documentation

### [balanced_selector_explanation.md](./balanced_selector_explanation.md)

Comprehensive documentation for the `core/balanced_selector.py` module, which provides:
- Balanced sampling of paper submissions for evaluation
- Stratified selection across desk rejection categories  
- Resume capability for interrupted evaluations
- Detailed function explanations and usage examples

**When to read this**: If you need to understand how the evaluation dataset selection works, or how to use the balanced evaluation features in the CLI.
